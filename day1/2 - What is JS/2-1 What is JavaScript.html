What is JavaScript?

The basics of web development is the combination of HTML, CSS, and JavaScript. HTML is the structure of the page, CSS is the styling/presentation, and JavaScript is the behavior layer.

JavaScript is what makes websites dynamic and interactive. Meaning we can:

  • Change the content of the page based on user input or other events
  • Create dynamic animations
  • Validate forms/input from the user
  • Create browser based video games
  • Create full fledged web applications (like Google Docs, Gmail, etc.)

Some more details on the language:

  • JavaScript was originally called "LiveScript" and was written in 10 days
  • It is dynamically typed, meaning you don't have to specify whether your variables are holding a string, or a number, etc. The language does it for you.
  • Was originally only intended to run on the clients browser
  • Now runs everywhere, for better or worse

Limitations of JavaScript

When JavaScript is running on a client's browser, it cannot access the user's file system, it also cannot reach into other webpages the user may have open.

JavaScript CAN modify the local computers file system when running inside Node.JS, but that is outside the scope of this course.

Versions of JavaScript

JavaScript has gone through many iterations, with new features and syntax being brought in over time. JavaScript is sometimes referred to as ECMAScript, or certain versions of JavaScript may be referred to specifically, for example ES6, ES7, etc.

We will go over some of these differences as we get further into the course.
